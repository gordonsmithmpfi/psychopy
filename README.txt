Projects in this repository
===========================

**continuousStims:** Stimuli that run continuously for several minutes. Based on Kalatsky & Stryker 2003. Rotating grating, contrast-phased grating, and moving bar.

**demos:** Examples of very simple stimuli. Try these out if you're new to PsychoPy.

**diagnostics:** Stimuli useful for measuring timing and calibration of your setup.

**episodicStims:** Stimuli that display for a few seconds, with inter-stimulus intervals. Grating, jitter-bar, and flashed bar.

**triggers:** For separating out triggering code from stim display code. 

**installers:** Drivers and instructions to help you get started.
