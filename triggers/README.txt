Triggering code
===============

Triggers are organized by the physical room they are used in. 

**examples:** Samples to help you get started. If you're making a new triggering class, use the template in this directory.

**Gordon2pt:** Triggering used in room 3225a, Gordon's 2-photon + intrinsic imaging setup.

**SLM:** Triggering used in room 3211, the SLM / Intrinsic / Ephys setup.
